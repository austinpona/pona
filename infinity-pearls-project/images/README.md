# Images Folder

## Instructions

Place all your product images in this folder.

### Required Images:

1. **logo.jpg** - Your Infinity Pearls logo (from the uploaded image with just the logo)

2. **Product Images** - Rename them as follows:
   - 1000803057.jpg - Aqua & Pearl Spider Charm
   - 1000803043.jpg - Red & Pearl 8-Ball Bracelet
   - 1000803054.jpg - Coral & Mint Bat Charm
   - 1000803048.jpg - Rainbow Flower Power
   - 1000803051.jpg - Pink & Black Heart Star
   - 1000803040.jpg - Purple Cross & Star
   - 1000803037.jpg - Infinity Link Bracelet
   - 1000803034.jpg - Pink Ombre Set
   - 1000803031.jpg - Crystal Daisy Bracelet
   - 1000803028.jpg - Pink Crystal Bear Charm

### Image Guidelines:

- **Format**: JPG, PNG, or WebP
- **Size**: Recommended 800x800px or larger
- **Aspect Ratio**: Square images work best
- **File Size**: Keep under 500KB for fast loading

### If Your Image Names Are Different:

Update the image paths in `js/script.js` in the `products` array to match your actual filenames.

Example:
```javascript
{
    id: 1,
    name: "Aqua & Pearl Spider Charm",
    description: "...",
    price: 60,
    image: "images/your-actual-filename.jpg"  // Update this
}
```
